# Support

## Documentation

- GonkaGate docs: https://gonkagate.com/en/docs
- Quickstart: https://gonkagate.com/en/docs/quickstart
- API reference: https://gonkagate.com/en/docs/api

## Questions and Troubleshooting

Open a GitHub issue:

- https://github.com/GonkaGate/gonkagate-examples/issues

When asking for help, include:

- example path (for example `sdk/node/openai-chat`)
- OS and runtime version
- exact command you ran
- full error message

## Bug Reports

Please include:

- steps to reproduce
- expected behavior
- actual behavior
- logs/output (with secrets removed)

## Security Issues

For vulnerabilities, follow `SECURITY.md` and do not open public issues.
