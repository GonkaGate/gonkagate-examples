# GonkaGate Production-Style Examples

`examples/` contains runnable integration patterns built on top of the GonkaGate OpenAI-compatible API.

## Why these examples

- Show practical app-level patterns, not only minimal API calls.
- Demonstrate streaming UX, agent workflows, and runtime handling.
- Provide reusable starters for production integration projects.

## Available Examples

- [Next.js Vercel AI SDK Chat](nextjs-vercel-ai-sdk-chat/README.md)
- [Cross-Platform CLI Chat (Go)](gonkagate-chat-cli/README.md)
- [Agent Examples](agents/README.md)

## Quick Start Path

1. Choose an example folder (or an agent example under `agents/`) and open its README.
2. Install dependencies.
3. Create `.env` from `.env.example`.
4. Run smoke mode if available.
5. Run the app or script.
