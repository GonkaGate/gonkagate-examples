# GonkaGate SDK Quickstarts

`sdk/` contains language-specific quickstarts for GonkaGate OpenAI-compatible API integration.

## Why these SDK examples

- Validate API key and model configuration quickly.
- Test chat and streaming behavior in popular SDKs.
- Keep onboarding copy-paste friendly for first-time setup.

## Quick Start Path

1. Open one example folder.
2. Install dependencies from that folder README.
3. Configure `.env` from `.env.example`.
4. Run smoke mode first, then a real request.
